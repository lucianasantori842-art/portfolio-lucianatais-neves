# Contribuindo para o Portfólio

Obrigada pelo interesse em contribuir para este portfólio! Este documento fornece diretrizes e instruções para contribuições.

## Como Contribuir

### Reportando Bugs ou Sugestões

Se encontrou um erro ou tem uma sugestão de melhoria:

1. Verifique se o problema já foi reportado
2. Descreva o problema com clareza
3. Forneça exemplos específicos para demonstrar os passos
4. Descreva o comportamento observado e o que você esperava

### Sugerindo Melhorias

Sugestões são sempre bem-vindas! Para sugerir uma melhoria:

1. Use um título claro e descritivo
2. Forneça uma descrição detalhada da sugestão
3. Liste alguns exemplos de como a melhoria seria útil
4. Mencione outras soluções similares que você conhece

## Diretrizes de Estilo

### Markdown

- Use títulos apropriados (H1, H2, H3)
- Mantenha linhas com no máximo 80 caracteres
- Use listas com formatação consistente
- Inclua links quando apropriado

### Conteúdo

- Seja claro e conciso
- Use linguagem profissional
- Evite jargão desnecessário
- Forneça exemplos quando relevante

## Processo de Revisão

1. Submeta sua contribuição
2. Aguarde revisão
3. Responda a feedback ou sugestões
4. Após aprovação, sua contribuição será integrada

## Código de Conduta

Ao contribuir, você concorda em:

- Ser respeitoso com outros contribuidores
- Aceitar críticas construtivas
- Focar no que é melhor para a comunidade
- Manter um ambiente profissional e inclusivo

**Obrigada por contribuir!**
